── PALET WARNA ────────────────────────────────────────────────
  Background utama : #f5ead0  (krem hangat)
  Gold / aksen     : #d99a34
  Teks utama (ink) : #69755a  (hijau keabu-abuan)
  Garis dekorasi   : rgba(184, 141, 63, 0.86)

── TIPOGRAFI ──────────────────────────────────────────────────
  FONT SCRIPT (nama mempelai, judul dekoratif)
    → Nama font   : Playlist Script
    → Warna       : #d99a34 (gold)
    → Weight      : Regular (400)

  FONT BODY / META (semua teks keterangan)
    → Nama font   : GFS Didot  (fallback: Didot, serif)
    → Warna       : #69755a (ink)
    → Letter-spacing lebar (0.12em – 0.28em)

  PANDUAN UKURAN FONT (konversi dari clamp responsif):
    Judul script besar   → ±90–112px  (cover, halaman salam, dll)
    Angka tanggal        → ±72–96px   (bold, warna gold)
    Nama mempelai        → ±48–64px   (script)
    Teks keterangan      → ±14–20px
    Teks kecil / label   → ±10–14px   (huruf kapital, spasi lebar)
